# Инструкция по деплою WoodLines

## Для хостинга с поддержкой Node.js (Vercel, Netlify, Railway и т.д.)

1. Распакуйте архив
2. Установите зависимости: `npm install`
3. Соберите проект: `npm run build`
4. Запустите: `npm start`

## Для статического хостинга

1. Распакуйте архив
2. Установите зависимости: `npm install`
3. Экспортируйте статический сайт: `npm run build && npm run export`
4. Загрузите папку `out` на хостинг

## Быстрый деплой на Vercel

1. Зайдите на vercel.com
2. Импортируйте проект из GitHub: https://github.com/Gel788/WoodLines
3. Vercel автоматически определит Next.js и задеплоит

## Переменные окружения

Проект не требует переменных окружения для базовой работы.

## Структура проекта

- `app/` - страницы Next.js
- `components/` - React компоненты
- `public/images/` - изображения
- `.next/` - собранный проект (создается после `npm run build`)
